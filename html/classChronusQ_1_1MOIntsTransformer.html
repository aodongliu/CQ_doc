Detailed Description

template<typename MatsT, typename IntsT>
class ChronusQ::MOIntsTransformer< MatsT, IntsT >

Templated class to handle AO to MO integral transformation for both one-body and two-body terms.

WARNING: for 1C only alpha part of the MO (mo[0]) is used

main interface to transform G(InactD)

G(D)(p, q) = gdFC * \sum_{mu,nu} C(mu,p)^H * C(nu,q) *
\sum_{lm,sg,i} C(lm,i)^H * C(sg,i) * [(mu nu | lm sg) - xFC * (mu sg | lm nu)]}]

gdFC ... two body part contribution factor 1.0 for 2C/4C and 2.0 for 1C, xFC ... exchange contribution factor, 1.0 for 2C/4C and 0.5 for 1C

Parameters
[in]pert... external EM pertubations
[in]MOGD... resulted G(D) in MO basis
[in]coreIndex... core index for computing densities
[in]moType... mo indices for p and q, respectively
[in]deltaPQ... if index p and q are identical
[in]cacheAOGD... controls whether to cache AOGD, default false
[in]cacheId... used to identify caches, default none

main interface to transform Core Hamiltonian:

Major Intefaces to obtain different intgrals in MO basis See details in include/mointstransformer/impl.hpp

HCore(p,q) = \sum_{mu,nu} C(mu,p)^H * C(nu,q) * [h1e(mu,nu) optional{ + gdFC * \sum_{lm,sg,i} C(lm,i)^H * C(sg,i) * [(mu nu | lm sg) - xFC * (mu sg | lm nu)]}]

gdFC ... two body part contribution factor 1.0 for 2C/4C and 2.0 for 1C, xFC ... exchange contribution factor, 1.0 for 2C/4C and 0.5 for 1C

Parameters
[in]pert... external EM pertubations
[in]MOHCore... resulted HCore in MO basis
[in]moType... mo indices for p and q, respectively
[in]deltaPQ... if index p and q are identical
[in]coreIndex... include core contributions if specified with core index char otherwise MOHCore will just be H1e, default null char

main interface to transform TPI

(p q | r s) = \sum_{mu,nu,lm,sg} C(mu,p)^H * C(lm,r)^H * [(mu nu | lm sg) - xFC * (mu sg | lm nu) ] * C(nu,q) * C(sg,s)

xFC ... exchange contribution factor, 1.0 for 2C/4C and 0.5 for 1C

Parameters
[in]pert... external EM pertubations
[in]MOTPI... resulted TPI in MO basis
[in]moType... mo indices for p, q, r and s, respectively
[in]cacheIntermediates... controls whether to cache AO integrals for InCore N5 and cache half transformed integrals for InCore N6 and DIRECT N6
[in]withExchange... whether compute Exchange part of the integrals
[in]delta... whether two or more indice are indentical see options at include/mointstransformer.hpp
