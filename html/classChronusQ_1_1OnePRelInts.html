Detailed Description

template<typename IntsT>
class ChronusQ::OnePRelInts< IntsT >

Templated class to handle the evaluation and storage of one electron integral matrices O, pOp, and three pOxp in a finite basis set.

Templated over storage type (IntsT) to allow for a seamless interface to both real- and complex-valued basis sets (e.g., GTO and GIAO)
