Detailed Description

template<typename MatsT, typename IntsT>
class ChronusQ::TPIContractions< MatsT, IntsT >

Templated class to define the interface to perform transformations and contractions of ERInts. Handles the contraction of 2-body (3,4 index) integrals with 1-body (2 index) operators.

Templated over matrix type (MatsT) to allow for a seamless interface to both real- and complex-valued coefficients and density.
