Detailed Description

template<typename MatsT, typename IntsT>
class ChronusQ::WaveFunction< MatsT, IntsT >

The WaveFunction class. The typed abstract interface for all classes which admit a well defined wave function (HF, KS, etc).

Adds knowledge of storage type to WaveFunctionBase

Specializes the Quantum class of the same type;
