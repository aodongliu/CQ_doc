template<typename T >
T ChronusQ::CQInputFile::getData ( std::string  )

Template function which returns the value of a data field from the input file in a specified datatype given a formatted query string.

i.e.

Input entry: [SCF] DENTOL = 1E-6

Query double tol = input.getData<double>("SCF.DENTOL");

This example returns the value of the string data field "SCF.DENTOL" as a double precision number. Various specializations of this function exist for various datatypes

Parameters
[in]sFormatted query string to be parsed
Returns
