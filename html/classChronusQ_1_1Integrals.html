Detailed Description

template<typename IntsT>
class ChronusQ::Integrals< IntsT >

Templated class to handle the evaluation and storage of integral matrices representing quantum mechanical operators in a finite basis set.

Templated over storage type (IntsT) to allow for a seamless interface to both real- and complex-valued basis sets (e.g., GTO and GIAO)

Real-valued arithmetics are kept in AOIntegrals
