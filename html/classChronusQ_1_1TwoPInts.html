Detailed Description

template<typename IntsT>
class ChronusQ::TwoPInts< IntsT >

Templated class to handle the evaluation and storage of electron-electron repulsion integral tensors in a finite basis set.

Templated over storage type (IntsT) to allow for a seamless interface to both real- and complex-valued basis sets (e.g., GTO and GIAO)
