Detailed Description

template<typename MatsT, typename IntsT>
class ChronusQ::MCWaveFunction< MatsT, IntsT >

The MCWaveFunction class. The typed abstract interface for all classes for which the wave function is described by a single slater determinant (HF, KS, PHF, etc).

Adds knowledge of storage type to MCWaveFunctionBase
