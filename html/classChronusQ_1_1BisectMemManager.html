Detailed Description

template<typename T>
class ChronusQ::BisectMemManager< T >

The BisectMemManager class A memory manager class specially designed for computing Cholesky decomposition pivots. When the memory requirement of each vector is less than half of the original, number of available memory vectors double.

Warning
Invalid for memVecLen > 2^32
