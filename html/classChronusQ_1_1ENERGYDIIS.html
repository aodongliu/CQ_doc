Detailed Description

template<typename T>
class ChronusQ::ENERGYDIIS< T >

The Energy DIIS class. A class to perform a DIIS interpolation based on a series of energies and couplings.

In general it solves the equation energy*c - 1/2 c^T*B*c where c are a set of coefficients between 0. and 1. This object uses a set of optimization variable that maintain this constraint, which allows for unconstrained optimization of the function.

This object inherits the BFGS optimizer to find the
minimum for the coefficients.
