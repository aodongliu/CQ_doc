Detailed Description

The BasisSet struct. Contains information pertinant for a gaussian type orbital (GTO) basis set.

Precomputes useful intermediates relating to the basis set as well as stores useful constants (nBasis, nPrimitive) and maps (maps shell # -> basis function #, etc). Utilizes a storage format consistent with the Libint2 integral package (libint2::Shell)
