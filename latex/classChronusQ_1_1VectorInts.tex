\subsubsection*{template$<$typename IntsT$>$\newline
class Chronus\+Q\+::\+Vector\+Ints$<$ Ints\+T $>$}
Templated class to handle the evaluation and storage of dipole, quadrupole, and octupole integral matrices in a finite basis set. 

Templated over storage type (IntsT) to allow for a seamless interface to both real-\/ and complex-\/valued basis sets (e.\+g., GTO and GIAO)
