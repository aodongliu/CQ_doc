\doxysubsection{Detailed Description}
The \mbox{\hyperlink{structChronusQ_1_1BasisSet}{Basis\+Set}} struct. Contains information pertinant for a gaussian type orbital (GTO) basis set. 

Precomputes useful intermediates relating to the basis set as well as stores useful constants (n\+Basis, n\+Primitive) and maps (maps shell \# -\/\texorpdfstring{$>$}{>} basis function \#, etc). Utilizes a storage format consistent with the Libint2 integral package (libint2\+::\+Shell)
