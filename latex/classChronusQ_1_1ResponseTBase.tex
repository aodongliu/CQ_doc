\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Chronus\+Q\+::\+Response\+TBase$<$ T $>$}
Type dependent interface for Response classes. 


\begin{DoxyItemize}
\item Defines problem dimension / (optional) matrix storage
\item Stores the response (RESIDUE / FDR) results
\item Provides access to memory manager
\item Defines the preconditioner
\item Allocates memory
\item General \char`\"{}run\char`\"{} functions for response calculations
\item Provides pure virtual interface for method specific specializations
\begin{DoxyItemize}
\item Form full matrix
\item Form RHS
\item Configuration options (set up)
\item Normalizing eigenvectors
\item Forming linear transformations 
\end{DoxyItemize}
\end{DoxyItemize}
