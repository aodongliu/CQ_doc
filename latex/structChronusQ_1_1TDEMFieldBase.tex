\doxysubsection{Detailed Description}
Base struct for the specification of an electromagnetic field perturbation. 

Provides a minimal structure to build a \mbox{\hyperlink{structChronusQ_1_1TDEMField}{TDEMField}} specification knowing that the \mbox{\hyperlink{structChronusQ_1_1TDEMField}{TDEMField}} will all inherit \mbox{\hyperlink{structChronusQ_1_1EMFieldBase}{EMField\+Base}}.

All \mbox{\hyperlink{structChronusQ_1_1TDEMField}{TDEMField}} classes are derived from this one.

Obtain the cartesian field tensor at a specificed time. 

Pure virtual, to be specified by derived classes.

All derived classes are aware of some tensorial representation of the EM field perturbation, $ \hat{v} $. This function returns

\[
  v(t) = \hat{v} \cdot f(t)
\]

where $ f $ is the envelope function.
