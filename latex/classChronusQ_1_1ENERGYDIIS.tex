\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Chronus\+Q\+::\+ENERGYDIIS$<$ T $>$}
The Energy \mbox{\hyperlink{classChronusQ_1_1DIIS}{DIIS}} class. A class to perform a \mbox{\hyperlink{classChronusQ_1_1DIIS}{DIIS}} interpolation based on a series of energies and couplings. 

In general it solves the equation energy$\ast$c -\/ 1/2 c$^\wedge$\+T$\ast$\+B$\ast$c where c are a set of coefficients between 0. and 1. This object uses a set of optimization variable that maintain this constraint, which allows for unconstrained optimization of the function. ~\newline


This object inherits the \mbox{\hyperlink{classChronusQ_1_1BFGS}{BFGS}} optimizer to find the ~\newline
 minimum for the coefficients.
